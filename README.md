# VenturePilot AI

VenturePilot AI is an AI-powered startup planning assistant developed using IBM watsonx Orchestrate.

## Features

- Business Idea Validation
- Business Plan Generation
- Market Research
- Funding Suggestions
- Investor Readiness
- AI-powered Conversations

## Technologies Used

- IBM watsonx Orchestrate
- IBM Cloud
- Python

## IBM Project

The AI agent is deployed on IBM watsonx Orchestrate.
