# VenturePilot AI

VenturePilot AI is an AI-powered startup planning assistant built using IBM watsonx Orchestrate.

## Features

- Business Idea Validation
- Business Plan Generation
- Market Research
- Funding Suggestions
- Investor Readiness
- AI-powered Conversations

## Technologies Used

- IBM watsonx Orchestrate
- IBM Cloud
- Python
